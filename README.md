# math-final

NIPT的时点选择与胎儿的异常判定

## Q1：Y染色体浓度与孕周、BMI的关系建模
* 广义加性混合模型（GAM），以孕妇ID为随机效应，B样条拟合孕周与BMI的非线性主效应。

## Q2：基于BMI的最佳NIPT时点分组优化
* 动态规划 + 分布鲁棒优化（KL-DRO），结合Q1预测值计算失败概率，构建含重测与延迟风险的成本矩阵，求解最优BMI分段与时点。

## Q3：多维因素干预下的个性化NIPT时点方案
* 多维感知DRO + 标量最优化，将年龄异质性融入鲁棒预算，按个体求解最佳时点并用线性回归输出“基准时间 + 年龄斜率”的预测公式。

## Q4：女胎异常判定
* XGBoost + Platt校准。
* 构造多染色体特征与质量惩罚因子，引入单调性约束与分层分组交叉验证处理不平衡样本，最后在校准概率上以FPR≤0.10为硬约束搜寻最佳阈值。

## 代码结构
* `template.py`：数据预处理
* `q1.py`：GAMM建模
* `q2.py`：DP+DRO求解
* `q3.py`：年龄敏感的DRO优化与回归
* `q4.py`：XGBoost训练与校准评估
